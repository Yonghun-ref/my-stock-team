# my-stock-team

**종목명만 말하면, 애널리스트 5명이 대신 일해주는 Claude Code 플러그인입니다.**

재무를 보는 사람, 주가 흐름을 보는 사람, 뉴스를 보는 사람, 리스크를 정리하는 사람, 마지막으로 점검하는 사람 — 이 다섯이 협업해서 리서치 문서를 쓰고, 검증한 뒤, **KB 디자인의 PPT 리포트까지** 자동으로 만들어 줍니다.

> 💡 **Claude Code가 처음이라면?** Claude Code는 터미널(검은 명령어 창)에서 대화하듯 일을 시키는 AI 도구입니다. 이 플러그인을 설치하면, 거기에 "○○ 분석해줘"라고 한국어로 말하기만 하면 됩니다.

---

## 1. 설치 (딱 두 줄)

Claude Code 안에서 아래 두 줄을 순서대로 입력하세요.

```
/plugin marketplace add yonghun-ref/my-stock-team
/plugin install my-stock-team@my-stock-team
```

- 첫 줄: 이 플러그인이 있는 곳을 Claude Code에 알려줍니다.
- 둘째 줄: 실제로 설치합니다.

설치가 끝나면 바로 쓸 수 있습니다.

---

## 2. 써보기 (이게 전부입니다)

Claude Code에 이렇게만 말하세요:

```
율촌화학 분석해줘
```

그러면 팀이 알아서 순서대로 일합니다:

| 순서 | 누가 | 무엇을 |
|------|------|--------|
| 1 | 펀더멘털 · 시장/기술 · 뉴스/심리 애널리스트 (동시에) | 재무, 주가 추세, 최근 뉴스를 각자 조사 |
| 2 | 리스크 매니저 | 결과를 모아 핵심 리스크와 종합 의견 정리 |
| 3 | (자동 저장) | `reports/율촌화학.md` 문서로 저장 |
| 4 | 검증 애널리스트 | 출처·숫자·표현을 최종 점검 |
| 5 | report-pptx 스킬 | `reports/율촌화학.pptx` 발표용 슬라이드로 변환 |

**다 끝나면 `.md`(글)와 `.pptx`(슬라이드) 두 파일이 `reports/` 폴더에 생깁니다.**

> 이미 만든 리포트만 슬라이드로 바꾸고 싶으면: `율촌화학 PPT로 만들어줘`

---

## 3. 처음에 한 번만: DART API 키 넣기 🔑

재무·공시 분석(펀더멘털 애널리스트)은 금융감독원의 **DART**라는 공공 데이터를 읽어옵니다. 이걸 쓰려면 **무료 열쇠(API 키)**가 필요한데, **보안상 플러그인에는 키를 넣지 않았습니다.** 그래서 처음 쓰는 분은 본인 키를 한 번만 발급받아 넣어주세요. (1~2분이면 됩니다.)

**① 키 발급받기**
1. https://opendart.fss.or.kr 접속 → 회원가입(무료)
2. "인증키 신청/관리"에서 키 발급 → 영문+숫자로 된 긴 문자열을 복사

**② 키를 내 프로젝트에 넣기**
1. 작업하는 폴더 안에 `.env`라는 이름의 파일을 만듭니다.
2. 그 안에 아래 한 줄을 적습니다 (`발급받은_본인_키` 자리에 복사한 키를 붙여넣기):
   ```
   DART_KEY=발급받은_본인_키
   ```
3. **`.env` 파일은 절대 깃(GitHub)에 올리지 마세요.** (열쇠가 새어나가지 않게)

> ✅ 주가·추세, 뉴스·심리 분석은 **키가 없어도** 동작합니다. DART 키는 재무·공시 부분에만 필요합니다.

## 4. 한 번만: 필요한 프로그램 설치

데이터를 읽고 PPT를 만들 때 쓰는 Python 도구들입니다. 터미널에 한 줄 붙여넣으면 됩니다.

```
pip install python-pptx FinanceDataReader pykrx requests
```

## 자주 묻는 것

- **무슨 종목이든 되나요?** 국내 상장 종목이면 종목명만 말하면 됩니다.
- **결과 파일은 어디 있나요?** `reports/` 폴더 안에 `.md`(글)와 `.pptx`(슬라이드)로 저장됩니다.
- **숫자는 믿어도 되나요?** 모든 수치에 출처와 기준일이 같이 적힙니다. 출처가 없는 숫자는 일부러 싣지 않습니다.

## 폴더 구성 (참고)

- `agents/` — 펀더멘털 · 시장/기술 · 뉴스/심리 · 리스크 · 검증 애널리스트 5명
- `skills/report-pptx/` — 리서치 글(`.md`) → KB 디자인 PPT 변환
- `commands/research.md` — `/research <종목명>` 전체 흐름을 묶은 명령

---
